Project Description

My project for this summer will be in a subfield of Artificial Intelligence and Machine Learning, called Reinforcement Learning (RL), where agents learn how to interact with their enviornment, in order to maximize some reward. The main problem we are trying to solve in RL is how to learn how to learn using a hierarchical structure, specifically how transfer between two related problems can be achieved by learning and reusing components of subproblems in each individual task. The goal of this project is to understand transfer in the context of hierarchical reinforcement learning and publish our findings.
